# Minz Racer

A 3D browser-based racing game built with Three.js. Race through a dynamic city environment, avoid traffic, and set high scores.

**Play Now:** https://someshwar-prox.github.io/Minz/

## Features

- **Persistent User Profiles**: Your progress and high scores are saved permanently
- **Progressive Difficulty**: Game speed increases as you level up
- **3D Graphics**: Realistic car models, buildings with illuminated windows, and dynamic lighting
- **Cross-Platform**: Works on desktop and mobile devices
- **Touch Controls**: Mobile-optimized with on-screen buttons
- **Collision Detection**: Real-time obstacle avoidance gameplay
- **Score System**: Track your score, level, and personal best

## How to Play

### Desktop Controls
- W or Up Arrow: Accelerate
- S or Down Arrow: Brake
- A or Left Arrow: Turn Left
- D or Right Arrow: Turn Right

### Mobile Controls
- Use on-screen touch buttons for acceleration, braking, and steering

## Gameplay

1. Enter your username to create or load your profile
2. Avoid obstacle cars while driving through the city
3. Score points by overtaking vehicles
4. Each 100 points advances you to the next level
5. Speed and difficulty increase with each level
6. Game ends on collision - try to beat your high score

## Technical Details

**Built With:**
- Three.js (r128) for 3D rendering
- Vanilla JavaScript
- HTML5 Canvas
- LocalStorage for data persistence

**Graphics Features:**
- Shadow mapping
- Fog effects for depth
- Metallic car materials
- Dynamic point lights
- Emissive materials for lights and windows

**Performance:**
- Optimized object pooling
- Efficient collision detection
- Smooth camera interpolation
- Responsive design for all devices
